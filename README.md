LanguageEx which is primarily known as VOXY, is a text based communication website where one can search people on the basis of seeking and offering langauge. They can talk to them as per their langauge and interest etc.

Features:-
1)Language Matching – Connect users based on selected languages.
2)Real-time Chat – Smooth and responsive text-based messaging.
3)User Profiles – Set language preferences, common interests and location.
4)Authentication – Secure login/signup system.

Tech Stack:-
1)Frontend: React
2)Backend: Node.js / Express 
3)Database: PostgreSQL
4)Auth: JWT
5)Real-time messaging: Socket.io

Future Improvements:-
1)Video calls
2)Match suggestions using AI
3)Progress tracking for each language
4)Community rooms
5)Time based practice
