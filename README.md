
# LanguageEx (formerly **VOXY**)
LanguageEx is a text-based communication platform where users can connect with others based on the languages they **seek** to learn and the languages they can **offer**. It helps people find meaningful language partners through shared language goals, interests, and location.

## Features

1. **Language Matching**
   Connect with users based on the languages they want to learn and the languages they can teach.
2. **Real-time Chat**
   Smooth, responsive, and reliable text-based messaging powered by Socket.io.
3. **User Profiles**
   Each user can set:
   * Language preferences (seeking/offering)
   * Common interests
   * Location
4. **Authentication**
   Secure login/signup system using JSON Web Tokens (JWT).

## Tech Stack

* **Frontend:** React
* **Backend:** Node.js / Express
* **Database:** PostgreSQL
* **Auth:** JWT
* **Real-time Messaging:** Socket.io
